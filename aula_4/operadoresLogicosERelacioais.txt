== igualdade 
0 == 0 -> True

Para analizar o valor e o tipo de dados adicionamos mais um sinal de =

"0" === 0 -> False

!= Diferente
0 != 0 -> False

"0" != 0 -> False
"0" !== "1" -> False

> Maior
1 > 2 -> False

< Menor
1 < 2 -> True

>= Maior ou igual
1 >= 2 -> False
2 >= 2 -> True

<= Menor ou Igual
1 <= 2 -> True
2 <= 2 -> True
